## Instalation 

1. Clone repository
```shell
git clone https://github.com/RasheedWpp/Frontend_HR_Dashboard.git
```
2. Install dependencies via npm or yarn
```shell
npm install
```
```shell
yarn
```

## Quick start
Run development server
```shell
yarn run
```
